# Cursor Control 1.0.0

Initial public release of Cursor Control for macOS.

## Includes

- Keyboard-driven cursor movement with H/J/K/L
- Universal scrolling with Control-H/J/K/L
- Left click, double-click, right-click, and Shift-drag support
- Configurable global shortcuts
- Configurable cursor movement keys
- Scroll distance, repeat, and acceleration settings
- Cursor speed and acceleration settings
- Menu bar app experience with onboarding and settings
- Free ad-hoc signed ZIP distribution
- Homebrew cask installation through `syedhy/cursor-control`

## Install

### Homebrew

```bash
brew tap syedhy/cursor-control
brew trust --cask syedhy/cursor-control/cursor-control
brew install --cask syedhy/cursor-control/cursor-control
```

### GitHub Release ZIP

1. Download `CursorControl-1.0.0-macOS.zip` from the assets below.
2. Unzip it.
3. Move `Cursor Control.app` to the Applications folder.
4. Open `Cursor Control.app`.
5. If macOS blocks the app, open System Settings > Privacy & Security and click Open Anyway for Cursor Control.
6. Grant Accessibility access when prompted.
7. Use the menu bar icon to open the guide, settings, or quit the app.

## Note

This app is currently ad-hoc signed and not Apple-notarized because it is
distributed without an Apple Developer Program account. macOS may require
approval from System Settings > Privacy & Security on first launch.

Only install Cursor Control from this GitHub Release or the official Homebrew
tap.
